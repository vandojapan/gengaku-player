1. Validate existing SPEC_KNOWN behavior with target-specific fixtures: `3007`, `3008`, `3015`, `3020`, `3026`.
2. Capture Maniacs reference logs for unresolved behavior: `3018`, `3021` subcommand `3`, `3025`, `3032`.
3. Implement only confirmed missing behavior:
   - `3018` operation `1`, once identified.
   - `3021` pixel-info subcommand `3`.
   - `3025` dispatch and edit-picture behavior.
   - `3032` dispatch and zoom behavior.
4. Re-run the target flow that creates result images and compare screenshots/PNGs pixel-for-pixel.
